# cultivator-bot

A Discord bot for all your favorite Cultivator catch phrases

# Usage

0. Download and install [git](https://git-scm.com/downloads), [python 3](https://www.python.org/downloads/),
   and [PDM](https://pdm.fming.dev/)
1. `git clone https://github.com/SwiftWinds/cultivator-bot.git`
2. `pdm install`
3. Follow instructions in `.env.template`
4. `pdm run python main.py`
